# Changelog

All notable changes to this project will be documented in this file.

## [7.0.0] - 2025-01-01

### Added

- Initial release of Vite template for Gasket
- React 18 with TypeScript support
- Vite for fast HMR and build tooling
- Express server integration via @gasket/plugin-express
- Basic testing setup with Vitest
- Minimal dependencies for quick startup
